Ahora tenemos que seguir con la siguiente fase. Debemos pasar el diseño a estructura HTML5 estándar y CSS3 para dar estilo a la estructura y los contenidos.

Esta fase tendrá las siguientes características:
Estructura semántica. Debéis usar para la estructura de la web elementos como: header, footer, nav, section...(apartado 2.3 de la teoría).
Crear estados en la navegación para que al pasar el ratón sobre los menús (Grupo 0+, Grupo 1,...) estos pueden cambiar de color o resaltarse. Además cuando estemos en cada una de las secciones esta deberá quedar claramente señalizada como podéis ver en el boceto que se adjunta.
Debe de ser compatible con diferentes navegadores (CSS Reset).
Optimizar la accesibilidad de nuestra web y usar textos alternativos que hagan más útil el lector de pantalla. Todas las imágenes deben tener un texto alternativo y los enlaces que necesiten descripción.
Utilizar medidas relativas. Se valorará muy positivamente el uso de media queries para modificar el contenido en función de la resolución del navegador, pero esto no será obligatorio.
Como recomendación, os dejamos especificados un tipo de estructura para el diseño y que tener en cuenta al empezar el proyecto. Así mismo, os dejamos una carpeta con las imágenes tipo de nuestra web.
